# Java Pattern Programs

## 📌 Overview

This repository contains Java pattern programs that help build a strong understanding of nested loops, conditional statements, and problem-solving skills. These programs are useful for beginners and for preparing for Java coding interviews.

---

## 📂 Patterns Included

### ⭐ Star Patterns
- Rectangle
- Hollow Square
- Right Angle Triangle
- Inverted Right Angle Triangle
- Reverse Left Half Pyramid
- Triangle Star Pattern
- Pyramid
- Rhombus
- Number Rhombus
- Parallelogram
- Butterfly Pattern
- Right Pascal Triangle

### 🔢 Number Patterns
- Number Pattern I
- Number Pattern J
- Number Increasing I
- Number Increasing J
- Number Increasing Reverse Pyramid
- Number Changing Pyramid
- Number Triangle
- Reverse Number Triangle
- Number Series
- Number Series 2

### 🔰 Special Patterns
- Palindrome Triangle Pattern
- 0-1 Triangle Pattern
- K Pattern
- Row × Column Pattern
